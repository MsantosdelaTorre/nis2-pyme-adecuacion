# Adecuación a NIS2 y ENS — Caso práctico: DentaCare

Caso práctico de adecuación a la **Directiva NIS2** y al **Esquema Nacional de Seguridad (ENS)** para una PYME del sector salud, aplicando la metodología **MAGERIT** para el análisis de riesgos.

> Proyecto formativo. La organización "DentaCare" es ficticia y los datos no corresponden a ninguna entidad real.

---

## Objetivo

Documentar el proceso completo de adecuación normativa de una PYME, desde la categorización del sistema hasta la definición de políticas y KPIs, siguiendo las metodologías oficiales aplicables en España.

## Contexto

DentaCare es una clínica dental ficticia con 15 empleados que trata datos de salud y, por tanto, es:

- **Entidad importante** según la Directiva NIS2 (sector salud).
- Sujeta al **RGPD** y la **LOPDGDD** por el tratamiento de datos de categoría especial.
- Aplicable el **ENS de categoría Media** al prestar servicios al sector público (mutua sanitaria).

## Metodología

- **MAGERIT** para el análisis y gestión de riesgos.
- **Real Decreto 311/2022** (ENS) para la categorización del sistema y selección de medidas.
- **Directiva (UE) 2022/2555 (NIS2)** para identificar obligaciones de la entidad.
- **Guías CCN-STIC** del Centro Criptológico Nacional como referencia de buenas prácticas.
